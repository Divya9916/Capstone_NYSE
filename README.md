# NYSE Stock Market Analysis – Data Analytics Capstone
## Project Overview

This project analyzes NYSE stock exchange data to uncover trends, patterns, and actionable insights for business and financial decision-making. It combines SQL, Python,and Power BI to perform end-to-end data analysis.

## Business Objective
Identify top-performing sectors and companies
Analyze revenue, profit, and growth trends
Support investment decision-making
Provide interactive dashboards for stakeholders

## Tools & Technologies
SQL – Data extraction, joins, aggregations
Python (Pandas, NumPy, Matplotlib, Seaborn) – Data cleaning & EDA
Excel – Initial data exploration
Power BI – Dashboard & visualization
DAX – KPIs and calculated measures

## Key Analysis Performed
1. Data Cleaning & Preparation
  Handled missing values and inconsistencies
  Standardized column formats
  Merged datasets (fundamentals + securities)

2. Exploratory Data Analysis (EDA)
  Revenue and profit trends over years
  Sector-wise performance comparison
  Identification of outliers and anomalies

3. SQL Analysis
  Top companies by revenue
  Sector-wise aggregation
  Time-based trend analysis

4. Power BI Dashboard
  KPIs: Revenue, Profit, Growth %
  Sector-wise performance visuals
  Trend analysis using line charts
  Interactive filters for business users

## Key Insights
Certain sectors consistently outperform others in revenue growth
A few companies contribute disproportionately to total market value
Market trends show cyclical patterns, useful for forecasting
Data-driven dashboards improve decision speed for stakeholders

## Business Impact
Helps investors identify high-growth sectors
Enables companies to benchmark financial performance
Supports leadership in making strategic investment decisions

## Assumptions & Limitations
Dataset is historical and may not reflect current market conditions
Missing values were handled using standard techniques (may affect precision)
Analysis assumes data accuracy from source

## How to Use
Run SQL queries to extract required data
Use Python scripts for EDA
Open Power BI file for dashboard insights
Refer to presentation for summary insights
